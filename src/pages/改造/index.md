# 改造

Mods I've made. So far, one example: a Final Fantasy XI Lua library that
converts between Shift JIS and UTF-8 so that UI addons can correctly display
Japanese characters taken from the in-game text. Using it requires editing
whichever addon renders CJK characters to include the library.

[Download `conversion.lua`](/改造/FFXI/conversion.lua)
