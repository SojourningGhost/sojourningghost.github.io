# 我　吾
◎「あれ（吾）」

｢あれ（生れ）｣。動詞｢あれ（生れ）｣の連用形ということです。自己の発生（それによる存在）が対象化（言語化）された表現。自己が表現する自己。つまり一人称。「さ寝むとはあれは思へど…」（『古事記』歌謡２８）。

◎｢われ（我）｣

｢わあれ（我生れ）｣。私という生まれ、私という存在、の意。すなわち、自己を表現する一人称は｢わ（我）｣であり、｢わあれ（我生れ）→われ｣は、私という生まれ、私という存在、と｢わ（我）｣の存在が確かなこととして、確信性をもって、表現されたもの。これも一人称になります。二人称で言われることもありますが、それは私（相手が自称する私）としてあるもの、の意。｢須須許理（すすこり：人名）が醸（か）みし御酒（みき）にわれ酔ひにけり｣（『古事記』歌謡５０）。

◎この｢あれ（吾）｣や｢われ（我）｣は｢誕生｣という抽象概念が表現されているわけではありません。｢あれ（生れ）｣｢わあれ（我生れ）｣という動態を表現しています。つまり、人は（日本人・日本語人）はそういうたびに生まれ存在化しています。

# こそあど　の　れ
これらは現代語の「こそあど」言葉を挙げられたものでしょうが、実は古語にも「あれ」「なれ」「われ」「かれ」「これ」などの代名詞が存在しますので、これらを合わせて考える必要がありそうです。

ところで古語では「あ（吾）」「な（汝）」「わ（吾）」「か（彼）」「こ（此）」だけでも代名詞として使われます。従って「れ」は代名詞の語尾と考えられます。  
しかし、これではあまりに素っ気なくて回答になりませんね。そこであくまでも推測ですが、「古事記」の中に「故」という文字が使われていて、学者は「カレ」と読ませて接続詞として扱っています。この「故」は大野晋氏によれば「か（彼）あれ（ば）」だと説明されています。この説明を少々強引に持ち込めば、akemisennseiのおっしゃるように、「れ」は「あれ」から来ていると言えるのではないでしょうか。

# ら
_ら_　　　　　　　Ｒ音による情況表現はＮ音の連続感によっている。それは、たとえば、その昔｢子供なな｣と言っていたものが｢子供ら｣と言うようになった、というような意味ではない。そういう語感、語音発生動因として情況を表現するＲ音が生じているということである。この連続感が持続感となり、Ｎ音による客観的な認了感に持続感が生じ、この、客観的認了感の持続感がＲ音による情況表現であり、概念の連続感・持続感は複数を表現する。動詞語尾のＡ音化による情況表現はそのＡ音の全体感による主体的なものであるが、Ｒ音による情況表現は客観的なものである。→｢赤ら顔｣｢あちら・こちら｣｢子供ら｣その他。

# で（にて）
「で」は以下の様に出来てきた。
最初は格助詞の「に」と動詞のすの連用形の「し」と接続助詞の「て」とが其の共起に依って複合して接続助詞と云われる「にして」が出来た。
「にして」から動詞のすが落ち「にて」と略される様に成った。
「にて」が音変化し「で」に成った。
手段や場合を表す格助詞的用法も「だ」の連用形と云われる、断定・指定を表す助動詞的用法も「にして」に由来する。

上記から幾つかの結論が出る。
一つ、「である」は断定の二重表現である。

問題は学校文法での「に」に対する誤認である。学校文法では「にして」が「であって」を意味する場合は「に」が「なり」の連用形とされる。然し、此れはとんでもない誤りである。「なり」は本来、格助詞の「に」と動詞の「あり」とが複合し音変化した物で、其の連用形と云われる「に」は「なり」を「に」と「あり」とに分離した「にある」の、原状の格助詞の「に」に過ぎない。従って、「に」と云う「なり」の連用形が存在せず、「にして」の両方の用法は同じ語源を持っている。「して」自体を格助詞に分類する辞書も在るけれど其の語源も亦動詞のすの連用形と接続助詞のてとが複合した物なので全く同じだ。

更に、広辞苑第六版に「一説に、「し」を指定の働きの語とする」と有る様に、動詞のすに指定・断定の機能が備わっていると考えられる。此れは実は驚くべき事では無い。何故なら然う云う用法を日常的句にも見る。挙例すれば「気がする」や「匂いがする」など感覚に関わる句でも「○○な体をしている」など有り様を表す句でも「す」が「あり」に似た働きを発揮している。斯く云う使い方も有れば「にあり」の「あり」の断定的機能をも果たせるとしても全く不思議では無い。
## 参考
「にして」に由来する格助詞寄りの「にて」しか入らなかっ たのである。
「二断定の意を表す「にて」の一体化 三 補助動詞「あり」の共起」より

「其の年の祭には、これを専にてぞありける」
「宇治拾遺物語」より
此の文は「これを専に（し）て」と解釈せずは文法が成立しない。

# 形容詞の連用形と格助詞の「に」とに下接する「て」
「～くて」・「にて」は動詞のすの連用形の「し」と接続助詞の「て」とが複合した「して」から動詞が落ちた物と思われる。
詰まり、「～くて」・「にて」は元々「ずして」の様に、「～くして」・「にして」だった。

# 面目次第も無い
「面目次第もない」とは、世間に対する面目丸つぶれで【次の対処のやりよう】もないとの意味です。【ㅤㅤ】内の部分が「次第」に該当します。 「次第」とは、「式次第」とか「成りゆき次第」などと使われているように「順番／順次」「次のステップ」などの意味もあります。 「面目次第もない」の場合においては、「面目が丸つぶれになってしまい、次にしなければならない対処の方法が無い」と言った使い方になります。 ※「お詫びの言葉もありません」という表現とどこか通じる物を感じます。
